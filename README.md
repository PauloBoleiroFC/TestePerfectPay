
## Teste Perfect Pay

## Para executar o teste

- Configure o banco de dados no arquivo .env

Execute os comandos:
- php config:clear
- php config:cache
- php cache:clear
- composer update
- php artisan migrate

- php artisan serve

## Após executar
faça o cadastro de um usuário para conseguir logar-se
